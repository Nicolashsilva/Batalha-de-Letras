# Batalha-de-Letras
Projeto Final LAPA/2025

Descrição:

O batalha de letras, é um projeto da faculdade da matéria de Laboratório de Algoritmos e Programação, durante este semestre foi estudado a linguagem C, devido a isso surgiu a necessidade de realizar um projeto final. O batalha de letras é uma adaptação moderna e dinâmica do famoso jogo da forca, desenvolvido em linguagem C. O projeto vai permitir estruturas para modos de multiplayer, campeonatos, eventos, recompensar conforma for passando de nível e demais estruturas que se veem necessária quando for ser estrturado. 
As funcionalidades do jogo consiste em 1 pessoa só, pode "escolher uma palavra" a partir de uma lista(poe exemplo: de animais), será mostrado o placar e o progresso do usuário.

As Bibliotecas utilizadas serão stdio.h, stdlib.h, string.h, time.h, podendo haver alteração de acordo com a necessidade. 

A tecnologia usada será C que foi a linguagem que usei o semestre todos para desenvolver e resolver as tarefas.
